# Metronome Practice Interval Timer

A lightweight practice timer with loopable practice/rest phases and a built‑in metronome.

## Features
- Looping practice/rest timer
- Metronome with selectable time signatures (1/4–7/8)
- Tempo control and independent volumes
- Auto-disable metronome during rest (optional)
- Tab title shows live time/phase while running

## Usage
1. Set Practice and Rest durations.
2. (Optional) Enable the metronome and choose time signature + tempo.
3. Press **Play** to start the loop.

## Controls
- **Play**: start or resume the timer
- **Pause**: pause the timer
- **Stop**: restart the current phase at its beginning (falls back to Practice)

## Notes
- Metronome uses audio-clock scheduling for tight timing.
- The metronome auto option disables clicks during rest.
